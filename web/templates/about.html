О проекте

Проект помогает получить информацию о гидрологической обстановке водохранилищ ГЭС России в графическом виде.

Техническая часть

Проект включает в себя 3 микросервиса:

  • Сборщик данных
  • Telegram бот
  • Веб-сайт ← вы находитесь здесь

Сборщик парсит сайты РусГидро и Енисейского БВУ, валидирует данные и складывает их в базу. В свою очередь, бот и сайт реализуют пользовательский интерфейс для построения графиков на основе сохраненных данных.

Используемые технологии:

  • Python 3.9
  • Django (сайт, админка, управление миграциями)
  • PostgreSQL (база данных)
  • Docker (контейнеризация)
  • Nginx (раздача статики, проксирование)
  • Aiogram (бот)
  • Asynpg (асинхронный доступ к БД)
  • Pydantic (валидация)
  • Matplotlib (графики бота)
  • Chart.js (графики на сайте)

С исходный кодом проекта можно ознакомиться в репозитории на GitHub.
